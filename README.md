## 🌌 The Vision
We are tired of passive learning. We are tired of coding courses that track no progress, offer no interaction, and put us to sleep. 

**Career Forge** was forged to solve this. It is a highly scalable, pure-frontend prototype built to demonstrate how gamification, artificial intelligence, and breathtaking UI/UX can completely revolutionize the way developers learn. 

No backend bottlenecks. No database crashes. Just infinite scalability and a flawless user experience.

---

## ⚡ Breathtaking Features

### 🗺️ Constellation Skill Maps
Paste **any YouTube playlist URL**, and watch the engine instantly transform the raw videos into a majestic, branching constellation map. Unlock nodes as you progress.

### ✨ AI-Generated "Wisdom Scrolls"
When you enter a node, our simulated AI agent rips the transcript and generates a custom **Wisdom Scroll**. It pulls out critical concepts, important notes, and generates an interactive Q&A section tailored exactly to that lesson.

### 📊 The Warrior's Dashboard
Every technique you master earns you **XP**. Track your completion rate, level up your rank from *Beginner* to *Master*, and challenge yourself with an epic **Boss Duel**.

### ⚙️ Bring Your Own Key (BYOK)
A stunning glassmorphism modal demonstrating scalable API architecture. Users can (mock) store their own OpenAI or Gemini API keys securely in local browser storage, bypassing rate limits entirely.

## 🚀 Run It Locally

Want to inspect the code and run the forge on your own machine? It takes less than 60 seconds:

```bash
# 1. Clone the repository
git clone https://github.com/shanmukh-dotcom/career_forge.git

# 2. Enter the forge
cd career_forge

# 3. Install dependencies
npm install

# 4. Ignite the server
npm run dev
```

> **Note:** This is a scalable frontend prototype. No actual API limits were harmed in the making of this app.
